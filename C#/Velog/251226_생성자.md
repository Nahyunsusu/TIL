# 생성자란?
생성자란 클래스 또는 구조체 인스턴스를 생성할 떄 런타임에서 호출되는 메서드이다. 클래스나 구조체에는 다른 인수를 사용하는 여러 생성자가 있을 수 있다. 
  먼저 클래스나 구조체 같은 객체가 인스턴스화되면 런타임은 해당 생성자를 호출한다. 생성자는 일반적으로 클래스와 이름이 같으며 새 객체의 데이터 멤버를 초기화한다.

## 매개변수 없는 생성자
  ```csharp
  using System;

public class Car
{
    public Car()
    {
        Console.WriteLine("자동차를 생성합니다.");
    }
    private int CarNum = 0000;
}


public class Solution
{
    public static void Main(string[] args)
    {
        Car car = new Car();
    }
}
```
위의 코드는 매개 변수 없는 생성자이다. Car라는 객체를 생성하기 위해서 Car의 생성자를 호출하였다. 함수를 호출하기 위한 방식이 두가지가 있다.
```csharp
Car car = new();
Car car = new Car();
```

## 매개변수가 있는 생성자
```csharp
using System;

public class Car
{
    public Car(string wantCarNum)
    {
        CarNum = wantCarNum;
        Console.WriteLine($"자동차를 생성합니다. \n 차 번호는 {CarNum}입니다.");
    }
    private string CarNum = 0000;
}


public class Solution
{
    public static void Main(string[] args)
    {
        Car car = new Car("2685");
    }
}
```
위의 코드는 매개변수가 있는 생성자이다. 객체를 생성할 때 매개변수를 넘겨주면 생성자 내부에서 해당 값을 받아 객체의 데이터 멤버(필드)를 초기화합니다.

## 생성자 오버로딩
```csharp
using System;

public class Car
{
    public Car()
    {
        Console.WriteLine($"자동차를 생성합니다. \n 차 번호는 {CarNum}입니다.");
    }

    public Car(string wantCarNum)
    {
        CarNum = wantCarNum;
        Console.WriteLine($"자동차를 생성합니다. \n 차 번호는 {CarNum}입니다.");
    }
    private string CarNum = "0000";
}


public class Solution
{
    public static void Main(string[] args)
    {
        Car car1 = new Car();
        Car car2 = new Car("2685");
    }
}
```
위의 코드는 연산자 오버로딩을 이용하여 생성자를 2개를 만든 경우이다. 객체를 생성할 때 인수를 넣지 않으면 자동으로 번호가 '0000'으로 적용되고 인수를 넣으면 그대로 번호가 그대로 적용된다.
