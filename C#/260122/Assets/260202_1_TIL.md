## Input System 핵심 구성 요소
![alt text](image-13.png)

### 구성 요소
1. Action Map
    - 입력을 상황 단위로 묶는 그룸
      - Player -> 캐릭터 조작
      - UI -> 메뉴 조작

2. Action
    - 하나의 행동을 의미
      - Move, Jump, Attack

3. Binding
    - Action을 실행하는 실제 입력 장치
      - WASD
      - Gamepad stick
      - Touch

4. Control Type
    - 실제 데이터 타입
      - Vector2
      - Button
      - float

## SendMessage
  - 붙어있는 오브젝트 안에서만 실행시켜줌
  

1. SendMessage/ Broadcast - 쓰지 않는걸 추천
2. Invoke Unity Events - 협업시 많이 활용된다.
3. Invoke C# Events - 캐릭터 이동, 전투 등 핵심 로직은 무조건 이 방식(개발자).
