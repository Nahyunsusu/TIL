## 배열이란?
 배열이란 동일한 타입의 변수를 모아놓은 데이터 집합이다. 배열을 사용하지 않는 다면 변수를 하나하나 선언을 해야된다.
 ```c#
    string player1;
    string player2;
    string player3;
    string player4;
    string player5;
    string player6;
 ```
 이것은 비효율적일 뿐만이 아니라 가시성도 좋지 않다는 것 이것들을 제어하기 위해서는 많은 동작들이 필요하다는 것이 문제가 된다. 이러한 문제들을 배열을 이용해 해결할 수 있게 된다.


 상당히 간편해 졌다. 그리게 배열에 접근할 때는 인덱스로 접근하는데 여기서 주의할 점은 인덱스는 항상 '0'으로 시작한다는 것이다.
 ```C#
    Console.WriteLine($"player={0}"); 
    Console.WriteLine($"player={1}"); 
    Console.WriteLine($"player={2}"); 
    Console.WriteLine($"player={3}"); 
    Console.WriteLine($"player={4}"); 
    Console.WriteLine($"player={5}");
    Console.WriteLine($"player={6}");
 ```
 일일이 변수를 선언하여 접근하는 것과는 다르게 배열의 인덱스를 이용한 접근은 상당히 간단하다

 아래는 배열의 선언 방법입니다.
 ```C#
 자료형[] 배열명 = new 자료형[크기];
 ```
  ```c#
    string[] players;
    string[] players = new string[num];
 ```
