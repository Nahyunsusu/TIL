# C# 연산자 및 식
C#은 여러 연산자를 제공한다. 이들 중 대부분 <span style="color:Aqua ">기본 제공 형식</span>에서 지원되며 이러한    형식의 값을 사용하여 기본 작업을 수행할 수 있다. 이러한 연산자는 다음 그룹을 포함한다.

1. 숫자 피연산자를 사용하여 산술 연산을 수행하는 산술 연산자 (+ - * /)
2. 숫자 피연산자를 비교하는 비교 연산자 (> < <= >=) 
3. 피연산자를 사용하여 논리 연산을 bool 수행하는 부울 논리 연산자 (! & | ^ && ||)
4. 정수 계열 형식의 피연산자를 사용하여 비트 또는 시프트 연산을 수행하는 비트 및 시프트 연산자 (~ >> << >>> & | ^)
5. 피연산자가 같은지 여부를 확인하는 같음 연산자 (==)

일반적으로 이러한 연산자를 오버로드 할 수 있다. 즉 사용자 정의 형식의 피연산자에 대한 연산자동작을 지정할 수 있다.

가장 간단한 C# 식은 리터럴과 변수 이름이다. 연산자를 사용하여 복잡한 식으로 결합할 수 있다. 연산자 우선 순위 및 결합성은 식의 연산이 수행되는 순서를결정한다. 괄호를 사용하여 연산자 우선 순위 및 결합성에 의해 부과되는 평가 순서를 변경할 수 있다.

```c#
int a, b, c;
a = 7;
b = a;
c = b++;
b = a + b * c;
c = a >= 100 ? b : c / 10;
a = (int)Math.Sqrt(b * b + c * c);
```
