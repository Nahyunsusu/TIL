# 조건문
## Selection문 - if, if-else 및 switch
1. if else 문
   ```csharp
   if (조건식) 
    {
        // 조건이 참일 경우 실행될 문장
    }
    ```
    이 예제의 조건식에서 true 또는 false를 반환하는 논리 연산자나 관계 연산자가 자주 등장한다. 조건식에 올 수 있는 건 true 또는 false의 값을 가지는 bool 형식이어야 한다. 예를 들어, 아래의 조건을 한번 살펴보자.

    ```csharp
    if (a > b) 
    {
        Console.WriteLine("a가 b보다 큽니다.");
    }
    ```
    a가 b보다 클 경우에 조건식은 참이 되고, 그 영역의 문장을 실행시킨다. 만약에 b가 a보다 크다면 거짓이 되어 이 조건문은 무시하고 넘어간다.

    참이 아닌 경우 부호를 반대로 바꿔줄 수도 있지만 참과 거짓을 한번에 쓰는 방법이 있는데 그 방법은 바로 <span style="color:Aqua ">'else'</span>이다.

    ```csharp
    if(a>b)
    {
        Consloe.WriteLine("a가 b보다 큽니다.");
    }
    else
    {
        Console.WriteLine("b가 a보다 큽니다.");
    }
    ```
    이렇든 else를 쓰면 참인 경우와 거짓인 경우 한번에 다룰수 있다. 하지만 이렇게 이분법적으로 다루지 않고 여러조건들을 걸어줄 수 있다면 어떨까? 그 경우는 <span style="color:Aqua ">'else if'</span>가 맡고있다.

    ```csharp
    if(a>b)
    {
        Consloe.WriteLine("a가 b보다 큽니다.");
    }
    else if(a==b)
    {
        Consloe.WriteLine("a와 b가 같습니다.");
    }
    else
    {
        Console.WriteLine("b가 a보다 큽니다.");
    }
    ```

    a,b 모두 정수라면 크기 비교는 이렇게 3가지밖에 존재하지않는데 if문 하나로 정리할 수 있다. else if는 한번 뿐이 아니라 여러번 사용될 수도 있다.

2. switch 문
   
   switch문은 if문과 달리 여러 개의 조건식의 결과를 검사할 때 사용된다. 기본적인 switch문의 형태는 아래와 같다.
    ```csharp
    switch (조건식) 
    {
    case 상수: // 만약 조건식의 결과가 이 상수와 같다면!
        // 실행될 코드
        break; // 탈출!
    case 상수:
        // 실행될 코드
        break;
    ...
    }
    ```
    이해를 돕기위해 예를 하나 들겠습니다. 만약에 조건식에 a % 2가 온다고 하고 a의 값이 13이며, 이 a가 홀수인지 짝수인지 구별하는 프로그램을 만들어 본다고 해보자.

    ```csharp
    switch (a % 2) {
    case 0:
        Console.WriteLine("짝수입니다!");
        break;
    case 1:
        Console.WriteLine("홀수입니다!");
        break;
    }
    ```

    이 경우에는 a를 2로 나눴을 때 나머지 값이 0이냐, 1이냐에 따라 나뉘는데 만약 0이라면 case 0:으로 이동하여 그 영역에 있는 문장을 실행시키고 break문을 만나 탈출한다. 1이면 case 1:로 이동하겠죠. 참고로, switch문은 정수 형식과 문자, 문자열 형식만을 지원한다는 것을 알아두시면 좋겠습니다.
