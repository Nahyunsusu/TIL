# 함수
- 지금까지는 모든 내용들을 main안에 한 줄씩 입력하였으나, 여러 줄이 모여 하나의 기능을 수행할 경우, 이를 묶어서 재활용 가능하다. 수학에서의 함수와 같이 프로그래밍에서 함수도 입력을 받아 해당 정보를 바탕으로 함수 내에서 하나하나 실행 후 , 결과물 하나를 도출해 내는 방식이다.

## 1. 함수의 개념 이해
예시를 들어보자
```csharp
using System;

namespace _1211_Temp
{
    public static class SwitchExample
    {
        static void Main(string[] args)
        {
            Console.WriteLine("높이값을 입력해주세요");
            int d1 = 0;
            int.TryParse(Console.ReadLine(), out d1);

            Console.WriteLine("너비값을 입력해주세요");
            int d2 = 0;
            int.TryParse(Console.ReadLine(), out d2);

            double area = (double)(d1 * d2) / 2;
            Console.WriteLine($"면적은 {area} 입니다.");
        }
    }
}
```
이런식으로 마름모의 넓이를 계산한다고 했을 때 main에 계산을 해놓게 된다면 다음에 또 넓이를 구하게 될 때 똑같은 내용을 반복해서 적어야 된다. 이를 해결하기 위해 여러 코드를 묶어서 결과를 도출해 내거나 행위를 지시할 때 함수를 사용한다. 이러한 코드 묶음을 '코드 블록'이라고 부르기도 한다. 반복되는 내용을 함수로 묶어 놓으면 필요할 때마다 꺼내서 써서 중복된 작업에서 일어날 수 있는 오류를 줄어들게 하여 유지보수가 좋아진다.

## 2. 함수의 4가지 형태 실전 활용
함수는 크게 4가지 종류로 구분할 수 있다.
1. <span style="color:pink ">리턴값</span>이 없고 <span style="color:pink ">매개변수</span>가 없을 때
2. <span style="color:pink ">리턴값</span>이 없고 <span style="color:Aqua ">매개변수</span>가 있을 떄
3. <span style="color:Aqua ">리턴값</span>이 있고 <span style="color:pink ">매개변수</span>가 없을 때
4. <span style="color:Aqua ">리턴값</span>이 있고 <span style="color:Aqua ">매개변수</span>가 있을 때
------------------------------

### 추가적인 반환값이 필요하다면? Out!
함수는 기본적으로 하나의 리턴값을 가진다. 그러나 Out키워드를 써줌으로써 추가적인 반환값을 받을 수 있다. 하지만 남용하면 함수에게 가에지는 부담이 커지고 코드 가시성이 떨어지기 때문에 남발하면 안된다.
```csharp
using System;
using System.Reflection.Metadata.Ecma335;

namespace _1209_Temp;

public class SwitchExample
{
    static void Main(string[] args)
    {
        Example example = new Example();

        Console.Write("첫번째 수 입력");
        int numa =int.Parse(Console.ReadLine());

        Console.Write("두번째 수 입력");
        int numb = int.Parse(Console.ReadLine());

        bool isBiggerThan0;
        int Sum = example.SumIsBiggerthan0(numa, numb, out isBiggerThan0);

        Console.WriteLine($"합: {Sum}");
        Console.WriteLine($"0보다 큰가?: {isBiggerThan0}");
    }
}

public class Example
{
    public int SumIsBiggerthan0(int a, int b, out bool IsBigger)
    {
        int sum = a + b;
        IsBigger = sum > 0;

        return sum;
    }
}
```
## 3. Ref, 인자값, 리턴값의 이해
```csharp
using System;
using System.Reflection.Metadata.Ecma335;

namespace _1209_Temp;

public class SwitchExample
{
    static void Main(string[] args)
    {
        Example example = new Example();

        Console.Write("첫번째 수 입력");
        int numa =int.Parse(Console.ReadLine());

        Console.Write("두번째 수 입력");
        int numb = int.Parse(Console.ReadLine());

        bool isBiggerThan0;
        example.SwapTwo(ref numa, ref numb);

        Console.WriteLine($"{numa} , {numb}");
    }
}

public class Example
{
    public void SwapTwo(ref int a,ref int b)
    {
        int temp = a;
        a = b;
        b = temp;
    }
}
```
